# N플리마켓 매크로 (Naver Fleamarket Macro)

네이버 플리마켓 상품 등록을 빠르고 편리하게 도와주는 크롬 확장 프로그램입니다.
자주 사용하는 상품 정보를 **템플릿으로 저장**해두고, 클릭 한 번으로 자동 입력할 수 있습니다.

---

## ✨ 주요 기능

### 1. 템플릿 관리
*   **저장**: 현재 작성 중인 상품 정보(제목, 카테고리, 가격, 설명 등)를 템플릿으로 저장
*   **로드**: 저장된 템플릿 목록에서 클릭 한 번으로 전체 필드 자동 입력
*   **미리보기**: 템플릿 목록에 마우스를 올리면 핵심 정보(가격, 상품명 등)를 포스트잇처럼 미리 확인
*   **관리**: 불필요한 템플릿 삭제 및 템플릿 이름 관리

### 2. 백업 및 공유 (Import/Export)
*   **내보내기 (Export)**: 특정 템플릿을 `.json` 파일로 저장하여 백업하거나 다른 사람과 공유
*   **가져오기 (Import)**: `.json` 파일을 불러와 내 목록에 추가

### 3. 사용자 경험 (UX)
*   **직관적인 UI**: 넓고 시원한 팝업 창 (600x600)
*   **안전한 실행**: 중복 클릭 방지 (Safe Lock) 및 실행 상태 알림 (Toast Message)
*   **스마트 스크롤**: 템플릿이 많아져도 헤더와 버튼은 고정되고 목록만 부드럽게 스크롤

---

## 🛠️ 설치 및 사용 방법

1.  이 저장소를 다운로드하거나 클론합니다.
2.  크롬 브라우저에서 `chrome://extensions` 접속
3.  우측 상단 **'개발자 모드'** 켜기
4.  **'압축해제된 확장 프로그램을 로드합니다'** 클릭 후 프로젝트 폴더 선택
5.  네이버 플리마켓 상품 등록 페이지(`fleamarket.naver.com/...`) 접속
6.  확장 프로그램 아이콘 클릭 후 기능 사용

## 📂 프로젝트 구조

```text
/
├── content.js          # 페이지 DOM 제어 및 데이터 입출력 (Main)
├── popup/              # 확장 프로그램 팝업 UI
│   ├── popup.html
│   ├── popup.css
│   └── popup.js
├── scripts/            # 기능별 로직 분리 (모듈)
│   ├── constants.js    # CSS 선택자 및 상수 관리
│   ├── utils.js        # 공통 유틸리티 함수 (Toast, Wait 등)
│   ├── extractors.js   # 데이터 추출(Get) 로직
│   └── setters.js      # 데이터 입력(Set) 로직
└── assets/             # 아이콘 등 정적 리소스
```

---

## ⚠️ 주의 사항

*   네이버 플리마켓 사이트의 구조(HTML/CSS)가 변경될 경우 동작하지 않을 수 있습니다.
*   이 경우 `scripts/constants.js` 파일의 선택자(Selector) 값을 수정하여 대응할 수 있습니다.
