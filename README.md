# Naver Fleamarket Macro

네이버 플리마켓 상품 등록 데이터를 기록하고 자동으로 입력하여 반복적인 등록 과정을 단축해 주는 매크로

---
## 기능
* **데이터 기록 및 자동 입력**: 이전 등록 데이터를 기반으로 필수 항목 자동 완성
* **카테고리 및 이미지 로드**: 카테고리 경로 자동 추적 및 이미지 업로드 지원
* **데이터 유지**: React 상태 동기화 로직을 통해 페이지 이동 시에도 입력값 보존

---
## 미구현 및 패스  (추후 개발)
* **브랜드/모델**: '없음'으로 일괄 처리 후 진행됩니다.
* **배송/지역**: 택배 상세 방법(집으로/편의점) 및 상세 직거래 지역 설정은 지원하지 않습니다.
